# Projeto Arduino pisca led

## Descrição
O objetivo deste projeto é mostrar a interação do arduino, através de suas portas digitais, com componentes eletrônicos externos, como o led e o resistor.

## Material necessário
- 1x Arduino;
- 1x protoboard;
- 1x led;
- 1x resistor de 470 ohms;
- 2x fios jumper;
